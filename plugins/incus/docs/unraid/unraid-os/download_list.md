Use this archive to find release notes and download links for Unraid OS versions. To update an existing server, use the built-in updater; see the [standard update process](/unraid-os/updating-unraid/#standard-update-process) for steps. To create a new boot device, use the [Unraid USB Creator](https://unraid.net/download).
The archive groups releases by support status. Only the latest patch or pre-release build in a supported minor series receives updates as appropriate; older builds in the same series are superseded. Releases marked **EOL** are not supported and no longer include updates.
If you are deciding which release to install, start with [Unraid OS release types](/unraid-os/updating-unraid/release-types/). For license eligibility details, see the [Licensing FAQ](/unraid-os/troubleshooting/licensing-faq/#no-extension).
